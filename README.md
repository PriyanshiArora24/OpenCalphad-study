## OpenCalphad-study
This website was developed as part of an exploratory project during the fourth semester at IIT (BHU), Varanasi to simplify OpenCalphad concepts through structured explanations and worked examples.

## Overview
The objective of this project was to simplify OpenCalphad for beginners by providing structured explanations, installation and setup guides, worked examples, and key observations in an easy-to-navigate web format.

## Features
- Structured learning content
- OpenCalphad installation and setup guide
- Gnuplot configuration guide
- Solved examples and key observations
- Clean and responsive user interface
- Easy navigation between topics

## Tech Stack
- HTML
- CSS
- JavaScript
